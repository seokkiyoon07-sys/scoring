# 📝 자동 채점 시스템

Streamlit 기반의 간단하고 직관적인 채점 프로그램입니다.

> 📚 **개발자를 위한 상세 문서**: [DEVELOPMENT_GUIDE.md](DEVELOPMENT_GUIDE.md) 참고

## 🚀 설치 방법

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

또는 개별 설치:

```bash
pip install streamlit pandas openpyxl
```

## 💻 실행 방법

터미널에서 다음 명령어를 실행하세요:

```bash
streamlit run app.py
```

브라우저가 자동으로 열리고 프로그램이 실행됩니다!

## 📋 파일 형식

### 학생 답안 파일 (CSV)

```csv
수험번호,과목코드,1번,2번,3번,4번,5번
2024001,MATH01,1,3,2,4,1
2024002,MATH01,1,2,2,4,1
2024003,ENG01,2,3,1,4,2
```

- 첫 번째 열: 수험번호
- 두 번째 열: 과목코드
- 세 번째 열부터: 문항별 답안

### 정답/배점 파일 (CSV)

```csv
과목번호,문항,정답,배점
MATH01,1,1,5
MATH01,2,3,5
MATH01,3,2,5
MATH01,4,4,5
MATH01,5,1,5
ENG01,1,2,10
ENG01,2,3,10
ENG01,3,1,10
ENG01,4,4,10
ENG01,5,2,10
```

- 과목번호: 과목 코드 (학생 답안의 과목코드와 일치해야 함)
- 문항: 문항 번호
- 정답: 해당 문항의 정답
- 배점: 해당 문항의 배점

## 📊 결과 파일 형식

채점 완료 후 다운로드되는 CSV 파일:

```csv
수험번호,과목코드,총점,만점,정답수,오답번호
2024001,MATH01,85,100,17/20,3, 7, 15
2024002,ENG01,90,100,18/20,5, 12
```

## ✨ 주요 기능

- ✅ CSV 파일 드래그 앤 드롭 업로드
- ✅ 과목코드별 자동 채점
- ✅ 점수, 정답수, 오답번호 자동 계산
- ✅ 평균, 최고점, 최저점 통계 제공
- ✅ 결과 CSV 파일 다운로드

## 🎯 사용 방법

1. 브라우저에서 프로그램 실행
2. 왼쪽 사이드바에서 "학생 답안 파일" 업로드
3. "정답/배점 파일" 업로드
4. "채점 시작" 버튼 클릭
5. 결과 확인 및 다운로드

## 💡 팁

- CSV 파일은 UTF-8 또는 CP949(EUC-KR) 인코딩 모두 지원됩니다
- 엑셀에서 "CSV UTF-8" 형식으로 저장하면 한글이 깨지지 않습니다
- 과목코드는 대소문자를 구분합니다

## 🛠️ 문제 해결

### 프로그램이 실행되지 않을 때

```bash
# Python 버전 확인 (3.8 이상 필요)
python --version

# 패키지 재설치
pip install --upgrade streamlit pandas
```

### 한글이 깨질 때

- 엑셀에서 저장 시 "CSV UTF-8 (쉼표로 분리)" 형식 선택
- 또는 메모장에서 UTF-8로 저장
